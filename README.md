# Game-List-CRUD
A simple CRUD using Express.js and mongoDB

## Installation

Create 'gamesList-app' mongodb database and import colletion 'game_feactures.json'
```
npm install
```

## Start the project

```
npm start || npm run dev
```
It will be running on localhost:5000
